Scoresheet — Lineup Parity

The Score tab reuses the exact same structure as the Lineup tab: round headers, then ringed Set cards (G1+G2 · G3+G4 · G5+G6) with the floating pill and "same time, two courts" hint. Same information, same shape — score entry just lives inside each game row.

The set ring reflects its games: teal when both confirmed, red if either score mismatches. Tap a box, type the score — rings and totals update live.
Each team enters its own copy of every score. Scores save automatically as you type, and a game confirms when both teams' numbers match — like trading paper scoresheets.
