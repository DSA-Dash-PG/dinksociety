Universal search — interactive mockup. One search box in the nav that finds any player or team across the league. This demo uses your real roster (it pulls live data on the site; a snapshot is built in so it works here too).

Try it: press ⌘K (or CtrlK), tap the search box, or just hit /. Type a name like “ryan”, “judy”, “smash”, or “kchn”.

This is a stand-in for any page. The search lives in the top nav and opens over whatever you’re looking at, so it works from anywhere on the site.

How it would work: the search icon sits in the global nav (and as an icon on mobile). Picking a player opens their profile at /player?name=…&team=…; picking a team opens /team?…. Results rank by best name match, then by DSR. Players show DSR + league rank; teams show their record. Scope chips let it be players-only or include teams.
